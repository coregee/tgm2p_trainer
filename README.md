# TGM2P Trainer

Bridge plugin + .exe for MAME to enable various modifiers for playing Tetris The Absolute The Grand Master 2 Plus.

## How to use

1. Download the [latest release](https://github.com/coregee/tgm2p_trainer/releases).
2. Move `/tgm2p-trainer` inside your MAME install's plugins directory.
3. Run `tgm2p-trainer.exe`, click the 'Config' button, and find your MAME .exe in the file browser.
4. Click "Launch MAME" to launch an instance of TGM2P with the trainer attached.
